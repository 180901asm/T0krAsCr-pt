# 🐍 Tokra Script
**Tokra Script** — **VF_Owner** と **Sladostrastnik** が作成した、[Fling Things and People](https://www.roblox.com/games/6961824067/) 用の最高の無料スクリプトです。
シンプルなインターフェースと、戦闘（HvH）に役立つ機能、そして楽しさを備えています。 長期間にわたり更新し続ける予定です。  

スクリプトに関するすべてのお知らせは、当社のDiscordコミュニティサーバーに掲載されます。  


# 📃 ツール
**Grab**: 15以上のモード、スーパースロー、さらに拡張;  
**Defense**: 新しいグッチアンチ、最高のアンチブロブマン、その他多数;  
**Loop**: 複数のターゲットに対して非常に高速なループ;  
**Blobman**: *近日公開... *;  
**Lag**: 他のプレイヤーのゲームプレイ機能を凍結する;  
**Auras**: 10人以上のプレイヤーと特別なオーラ;  
**Decoys**: NPCの追従、ジャンプ、フリング、所有;  
**Bombs**: プレイヤーを自動生成し、ループで爆発させる;  
**ESP**: プレイヤーモデルのハイライト;  
**Visual**: マップの色とグラフィックエディタ;  
**Whitelist**: カスタムホワイトリストとフレンドリストの設定;  
**Interface**: 高度なインターフェーステーマの設定;  
**Misc**: 美しい翼、オートファーム、プロットセーバー、その他。
